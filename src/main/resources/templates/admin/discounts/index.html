Как работает система скидок
  • Скидка применяется к комбинации: Тип пользователя + Тип товара
  • Тип пользователя задается в профиле пользователя
  • Тип товара задается при создании/редактировании товара
  • Если для комбинации нет скидки - применяется цена товара без скидки
  • Неактивные скидки (серые строки) не применяются
